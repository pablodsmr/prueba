Ejercicio 1. Estudio de distintos alojamientos Web
Hosting gratuito.
Ventajas:
  • Es gratis
  • Hay sitios profesionales para alojar un sitio sencillo de manera gratuita.
  • Puedes guardar archivos en ubicaciones diferentes para ocupar menos espacio.
Desventajas:
  • Puede existir publicidad forzada o insistencia para comprar servicios extra.
  • Es posible que no puedas monetizar tu sitio web.
  • Espacio de almacenamiento limitado.
  • Ancho de banda limitado.
  • Mayor vulnerabilidad y riesgos de seguridad.
  • Riesgo de compartir alojamiento con sitios spam.
Fuente
Hosting compartido.
Ventajas:
  • Precio económico
  • Soporte técnico
  • Instalación automática de las aplicaciones para desarrollo web.
Desventajas:
  • Exceso de consumo o malas intenciones por parte de otros usuarios
  • Poca personalización en configuraciones del servidor
  • Recursos limitados
  • Fallo de seguridad
Fuente
Hosting para revendedores.
Ventajas:
  • Es una excelente forma de emprender un negocio sin mucha inversión.
  • No requieren de atención constante.
  • Representa una inversión de poco riesgo, tanto para el revendedor como para el cliente.
Desventajas:
  • El margen de ganancia es bajo.
  • El revendedor no tiene contacto directo con los equipos.
  • Tanto el revendedor como el cliente dependen en gran medida del proveedor principal.
Fuente
Hosting VPS
Ventajas:
  • La libertad de un servidor dedicado
  • Administrar varios sitios criminal cuenta única
  • Combinan las características de alojamiento compartido y dedicado.
  • Se adapta tanto a medias como a grandes empresas
  • Flexibilidad
Desventajas:
  • Limitación de uso de CPU y RAM
  • Ancho de banda es limitado
  • El tiempo de procesamiento es mayor
  • Cliente tiene que controlar su servidor sino es administrado
Fuente
Hosting dedicados
Ventajas:
  • No comparte espacio ni capacidad, otorga gran independencia, pudiendo ser utilizado para multitud de usos.
  • Intuitivo panel de control
  • Seguridad
  • Disponen de un mantenimiento y actualización
  • Contratación de un servidor de alojamiento supone absoluta libertad
Desventajas:
  • Precio
  • Es recomendable tener conocimientos previos
Fuente
Cloud Hosting
Ventajas:
  • Flexibilidad o escalabilidad de los recursos
  • ofrecen la posibilidad de acceder al número de servidores que necesitemos de forma rápida y eficaz.
  • La mayoría de los proveedores de servicios de cloud hosting ofrecen paneles de administración o control muy fáciles de utilizar.
Desventajas:
  • Nuestra información están en manos de un proveedor que podría tener algún problema técnico.
  • Importancia de un proveedor de confianza
